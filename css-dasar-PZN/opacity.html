Opacity

  • Element di HTML bisa diatur transparasinya menggunakan property opacity
  • Perlu diperhatikan, saat menggunakan opacity, semua element baik itu background dan contentnya akan berubah menjadi transparan
  • Nilai opacity paling rendah 0 dan paling tinggi 1, kita juga bisa mengatur opacity dengan koma misalnya 0.8 atau .8 sama saja
