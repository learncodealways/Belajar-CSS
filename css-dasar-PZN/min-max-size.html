Min dan Max Size

  • Saat menambahkan size pada element, kadang kita ingin menentukan minila atau maksimal dari size element, hal ini untuk menjaga element tidak terlalu kecil atau terlalu besar
  • Kita juga bisa mengaturnya menggunakan property di CSS

Min Size

  • Untuk mengatur minimal kita bisa menggunakan property min-height untuk minimal tinggi
  • References-min-height
  • min-width untuk minimal lebar
  • References-min-width

Max Size

  • Untuk mengatur maksimal kita bisa menggunakan property
  • max-height untuk maksimal tinggi
  • References-max-height
  • max-widhth untuk maksimal lebar
  • References-max-width
