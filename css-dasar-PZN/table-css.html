Table

  • Saat kita membuat element table di HTML, kita akan banyak menggunakan Box Model

Table Border

  • Jika kita ingin mengatur border pada table, kita bisa menggunakan border pada table, tr, th, td dan lain-lain
  • References-border
  • Saat menggunakan border, kadang tiap kolom akan ada jarak, jika kita ingin menghilangkan jaraknya, kita bisa menggunakan border-collapse
  • References-border-collapse

Table Size

  • Untuk mengatur ukuran table, kita bisa menggunakan property width dan height
  • References-height
  • References-width

Table Alignment

  • Untuk mengatur posisi text pada tabel, kita bisa menggunakan property text-align atau writing-mode
  • References-text-align
  • References-writing-mode
