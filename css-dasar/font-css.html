Font

  • Saat membuat tulisan, biasanya Web Browser akan menggunakan default fontnya, tiap web browser memiliki default masing-masing
  • Saat membuat web baiknya kita menggunakan font yang sama sehingga tampilan web kita konsisten
  • Kita bisa mengatur font di halaman HTML menggunakan CSS dengan property font
  • References-Font

Font Family

  • Untuk mengubah jenis font, kita bisa menggunakan property font-family
  • Refrences-Font-Family
  • Menggunakan font-family sangat bergantung dengan sistem operasi yang digunakan, jika fontnya tidak ada di sistem operasi yang digunakan, maka hasilnya tidak akan sesuai dengan yang kita mau
  • Kita bisa menggunakan generic font family atau sering disebut fallback yang sudah menjadi standar untuk CSS
  • Generic-font-family

Font Style

  • Untuk mengubah style dari font, kita bisa menggunakan property font-style
  • References-Font-Style
  • Untuk mengubah ketebalan dari font, kita bisa menggunakan property font-weight
  • References-Font-Weight

Font Size

  • Untuk mengubah ukuran dari font, kita bisa menggunakan property font-size
  • References-Font-Size

Google Font

  • Salah satu penyedia font gratis yang bisa kita gunakan adalah Google Font
  • Kita bisa mencari font di Google Font, lalu menggunakna CSS untuk menambahkan font yang kita mau di halaman Web yang kita mau
  • Google-Font
