Text

  • CSS bisa digunakan untuk mengubah properties atau format untuk text
  • Selain colot, masih banyak yang bisa kita ubah dari properties atau format text

Text Alignment

  • Text alignment digunakan untuk mengubah rata tulisan, kita bisa menggunakan rule text-align, dimana memiliki beberapa nilai seperti
    1. left untuk rata kiri (default)
    2. right untuk rata kanan
    3. center untuk rata tengah
    4. justify untuk rata kanan dan kiri
  • References-text-align

Text Decoration

  • Text Decoration digunakan untuk menambah garis dekorasi ke text
  • Kita bisa menggunakan aturan text- decoration
  • References-text-decoration

Text Transformation

  • Text Transformation digunakan untuk mengubah uppercase dan lowercase text
  • Kita bisa menggunakan aturan text-transform
  • References-text-transform

Text Spacing

  • Text Spacing digunakan untuk mengatur jarak dalam text
  • text-indent digunakan untuk mengatur jarak di awal text : References-text-indent
  • leterr-spacing digunakan untuk mengatur jarak antar huruf : References-letter-spacing
  • line-height digunakan untuk mengatur jarak antar baris References-line-height
  • word-spacing digunakan untuk mengatur jarak antar kata References-word-spacing
  • white-space digunakan untuk mengatur bagaimana whitespace ditampilkan References-white-space

Text Shadow

  • Text Shadow digunakan untuk menambahkan efek bayangan pada text
  • References-text-shadow
