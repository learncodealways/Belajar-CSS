Combinator Selector

  • Combinator adalah sesuatu yang menjelaskan relasi antar selector
  • Terdapat 4 selector untuk combinator
    1. descendant selector (space)
    2. child selector(>)
    3. adjacent sibling selector(+)
    4. general sibling selector(~)

Descendant Selector

  • Descendant Selector adalah selector untuk memilih element anak dari element yang dipilih
  • Contoh Descendant Selector kode di bawah ini
            
            
            div p {
                color: red;
            }

  • Adjacent Sibling Selector digunakan untuk memilih satu element saudara setelah element yang dipilih
  • Sibling (saudara) element harus memiliki parent element yang sama
  • Contoh Adjacent sibling selector kode di bawah ini
            

            p + div {
