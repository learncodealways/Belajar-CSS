    Attribute Selector Tanpa Tag

    • Saat menggunakan attrubute selector, nama tag sebenanrya tidak wajib, jadi kita bisa langsung menggunakan atrubute selector
    • Contohnya : 
                      [tittle*="belajar"]{
                    color: red;
                }
    • Attribute Selector juga bisa digunakan pada Class atau ID
    • Contohnya : 
                      .article[title*="belajar"]{
                    color: red;
                }

                #ARTIKEL[title*="belajar"]{
                    color: red;
                    font-style: italic;
                }
