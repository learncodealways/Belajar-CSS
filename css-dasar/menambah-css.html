Menambah CSS ke HTML

  • Terdapat tiga cara untuk menambah CSS ke HTML
    1. Cara internal, dimana CSS disimpan di file yang sama dengan HTML. Yaitu dengan menggunakan tag style 
                      <style>
                      h1 {
                        color:red;
                      }
                </style>
    2. Cara external, dimana CSS disimpan di file yang berbeda dengan html. Yaitu menggunakan tag link
    3. Cara inline, dimana CSS disimpan dalam atribut styles pada tag html, cara ini tidak direkomendasikan
