Box Model

  • Saat menggunakan CSS, terdapat konsep bernama Box Model
  • Biasanya digunakan ketika ingin mengatur tata letak / layout pada element
  • Setiap element di HTML, memiliki Box yang terdiri dari content, padding, border dan margin
  • Reference-Box-Model
  • Gambaran Box Model
  • Penjelasan :
    1. Konten (Content): Ini adalah bagian utama dari elemen, seperti teks, gambar, atau elemen lainnya yang Anda letakkan di dalamnya
    2. Padding: Ini adalah ruang tambahan di sekitar konten, memberikan jarak antara konten dan batas elemen. Padding membuat elemen terlihat lebih teratur dan terpisah dari elemen lain di sekitarnya.
    3. Border: Ini adalah garis yang mengelilingi padding, memberikan batas visual antara elemen dan lingkungannya. Anda dapat menentukan warna, ketebalan, dan jenis garis border.
    4. Margin: Ini adalah ruang di luar border yang memisahkan elemen ini dari elemen-elemen lain di sekitarnya. Margin membantu mengatur tata letak dan jarak antar elemen.

    Jadi, secara singkat, box model membantu kita mengontrol dan mengatur tata letak serta tampilan elemen-elemen di halaman web. Dengan memahami konsep ini, Anda dapat mengelola ruang di sekitar elemen dan memastikan halaman web terlihat sesuai dengan yang diinginkan.
