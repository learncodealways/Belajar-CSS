Nilai CSS Display

  • inline
    • komponen ditampilkan secara inline (hanya mengambil tempat secukupnya)
  • block
    • komponen ditampilkan secara block (mengambil tempat kiri ke kanan komponen diatasnya)
  • inline-block
    • komponen ditampilkan secara inline, tapi kita bisa mengubah tinggi dan lebar komponennya
  • none
    • komponen akan dihapus dan tidak ditampilkan
